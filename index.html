Product Features

  • Proof of Ownership
  • Transfer of Ownership
  • Supply Chain
  • Theft/Recovery and Law Enforcement
  • Search Bikes and Owners
  • Maintenance Tracking

Technical Overview

At its core, BikeDeed uses Ethereum smart contracts based around the ERC721 standard for non-fungible tokens. File handling and hashing uses the IPFS (Interplanetary File System). Dynamic server functionality is handled by nginx and node.js. The frontend uses Javascript, HTML and Web3.

Product use Prerequisites

To use BikeDeed, you need a Web3 enabled browser such as Chrome with the Metamask plugin. For the iPhone or Android you should use the Trust or Cipher mobile wallets.
